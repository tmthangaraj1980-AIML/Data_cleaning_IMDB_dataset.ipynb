# End-to-End Data Cleaning and Preprocessing of a Messy IMDB Movie Dataset

This project focuses on cleaning and preprocessing a messy IMDB movie dataset using Python and Pandas.

## Key Cleaning Steps

- Handled missing values
- Removed irrelevant columns
- Fixed column names
- Corrected inconsistent data formats
- Converted data types
- Cleaned release year, duration, votes, income, and score columns
- Detected outliers using IQR method
- Saved the final cleaned dataset

## Tools Used

- Python
- Pandas
- NumPy
- Google Colab
